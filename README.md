# Nyancat

An Elixir port of the original nyancat CLI by Kevin Lange.

<img width="500" src="https://github.com/veelenga/bin/raw/master/nyancat.ex/demo.png" />

## Usage

```
$ git clone https://github.com/veelenga/nyancat.ex && cd nyancat.ex
$ mix deps.get
$ mix escript.build
$ nyancat
```
